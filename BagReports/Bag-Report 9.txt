#Bag-Report 9
Неверное отображение входных данных при сложении в результате.

Шаги:
1. Ввести в первое поле для ввода число "10000000000"
2. Ввести во второе поле для ввода число "5000000000000"
3. Нажать на кнопку "+"

Фактический результат: Выводит в поле для результата: "9999999827968.00 + 4999999913984.00 = 14999999217664.00".

Ожидаемый результат: Выводит в поле для результата: "10000000000.00 + 5000000000000.00 = 15000000000000.00".

Окружение: Anroid 9.0
